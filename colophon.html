Colophon

This website is made with accordance to Google HTML/CSS Style Guide with the following changes:

  • Indentation is done by tabs, not spaces.
  • Selectors are not separated by newlines.

It is written in en-x-amn, which means, more or less, crappy English.

This page is being written with sam text editor on laptop with Gentoo and Xubuntu Ubuntu and Windows 10 (the latter is actually not used by me).

This website is typeset in Noto Sans, headers are set in Lato, math and mathematical objects like trees are set in Neo Euler, and code is set in Source Code Pro.
